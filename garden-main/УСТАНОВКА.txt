ИНСТРУКЦИЯ ПО ЗАПУСКУ PLANTASIA
===============================

Шаг 1: УСТАНОВКА NODE.JS
-----------------------
1. Скачайте и установите Node.js с сайта: https://nodejs.org/
2. Выберите версию LTS (Long Term Support)
3. Следуйте инструкциям установщика
4. После установки ПЕРЕЗАГРУЗИТЕ компьютер

Шаг 2: НАСТРОЙКА ПРОЕКТА
----------------------
1. Откройте командную строку Windows
   - Нажмите Win+R
   - Введите "cmd" и нажмите Enter

2. Перейдите в папку проекта в командной строке
   - Например: cd C:\путь\к\папке\plantasia

3. Перейдите в папку backend
   cd backend

4. Установите зависимости
   npm install

Шаг 3: НАСТРОЙКА API КЛЮЧА
-----------------------
1. Создайте файл .env в папке backend:
   - Откройте Блокнот
   - Скопируйте следующий текст:
     PORT=3000
     PLANT_API_KEY=8uHFK9i5c3Oi0kpFeZ88Sg5x27FS67tR9lrGU0uRlFl60smh6Z
   - Сохраните как ".env"
   - При сохранении выберите "Все файлы (*.*)"
   - Убедитесь, что файл находится в папке backend

Шаг 4: ЗАПУСК ПРИЛОЖЕНИЯ
---------------------
1. В командной строке, находясь в папке backend, введите:
   node server.js

2. Вы должны увидеть сообщение:
   "Сервер запущен и доступен по адресу: http://localhost:3000"

3. Откройте браузер и перейдите по адресу:
   http://localhost:3000

4. Теперь вы можете использовать приложение!

РЕШЕНИЕ ПРОБЛЕМ
==============

Ошибка "node не является внутренней или внешней командой":
- Переустановите Node.js и убедитесь, что выбрали опцию "Add to PATH" при установке

Ошибка "Cannot GET /":
- Проверьте, запущен ли сервер и работает ли он без ошибок
- Убедитесь, что вы ввели правильный адрес в браузере

Растение не определяется:
- Используйте фотографию лучшего качества
- Убедитесь, что на фото хорошо видно растение
- Проверьте соединение с интернетом

Если у вас возникли другие проблемы, обратитесь за помощью или посмотрите подробную документацию в README.md
